=== v4 대비 핵심 변경사항 ===

1. 로컬 page_kind 선분류 (Gemini 호출 최소화)
   - local_classify(pno) 함수: 페이지 번호만으로 즉시 분류
   - p14~16: meta_explanatory → Gemini 호출 없이 처리
   - p17~53: toc             → Gemini 호출 없이 처리
   - p54~  : reaction_content → Gemini 호출 대상
   - 결과: 132장 중 약 40장 Gemini 절약

2. retry/backoff 정책 (유형별 구분)
   - Api503Error: 5초 → 15초 → 45초 backoff 후 재시도
   - TimeoutError: 10초 → 30초 후 재시도
   - RecitationError: 5초 후 재시도
   - 시도 순서: primary/original → primary/original → primary/fallback_prompt → fallback_model
   - 총 4회 시도 후 실패 시 fail_queue 등록

3. fallback model
   - 1차: gemini-2.5-pro (primary)
   - 4차: gemini-2.5-flash-preview-04-17 (fallback)

4. fail_queue 테이블 + CSV
   - fail_queue 테이블: page_no, failure_type, attempts, error_msg, batch
   - fail_queue CSV: 재처리 가능한 실패 목록
   - 실패 유형: api_503 / timeout / recitation / invalid_json / unknown

5. sample/full 모드 분리
   - --mode sample: 샘플 15장 (기본값, 검증용)
   - --mode full: 전체 132장 (GPT 승인 후)

6. SMILES confidence 임계값 강화
   - 0.85 → 0.90으로 상향
   - smiles_confidence < 0.90 → reactant_smiles=None, product_smiles=None

7. Title Case 정규화 (하이픈 처리 포함)
   - ACETOACETIC ESTER SYNTHESIS → Acetoacetic Ester Synthesis
   - SIMMONS-SMITH → Simmons-Smith
   - 2-aza-Cope → 2-Aza-Cope
